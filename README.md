## 文件组织


```
CodeForge/
├─ graph/           # 图论
├─ data-struct/     # 数据结构
├─ math/            # 数论 & 多项式
├─ string/          # 字符串
├─ dp/              # 动态规划
└─ misc/            # 杂项（快读、对拍、调试宏）
```


每个 `.cpp` 都是独立模板，直接复制即可塞进任意工程。

## 一键编译


```bash
g++ -std=c++23 -O2 -Wall -Wextra -o main filename.cpp && ./a.out
```


**提示**  
- 需要 C++23 及以上  
- 模板里如果用到 `__int128`，请确保评测机支持，否则自行改 `long long`
